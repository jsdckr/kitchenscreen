# piGathering
Barista and Ices Displays at Waimanako

Installation:
unzip to /home/projects/piGathering
As root:
cp -R piGathering/font/ubuntu /usr/share/fonts/ubuntu /usr/share/fonts/truetype
apt install python3-pyqt5
pip3 install pynput
pip3 install mysql.connector

Hopefully, all good from there!
